# Linux Labs

Repositório de estudos práticos em Linux e infraestrutura, com foco em
administração de sistemas, fundamentos de redes e preparação para ambientes
de nuvem.

## Objetivo
Documentar de forma organizada e progressiva os principais conceitos e
experimentos realizados em laboratório, priorizando entendimento técnico
e aplicação prática.

## Estrutura do repositório
- `00-fundamentos/` – conceitos básicos de Linux


## Metodologia
- Estudo teórico objetivo
- Execução prática em ambiente Linux
- Documentação clara e direta
- Evolução incremental dos temas

## Tecnologias
- Linux
- Shell
- Ferramentas de rede
- Containers (introdução)
- Cloud (camada posterior)
