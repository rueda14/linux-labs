# Fundamentos de Linux

Este diretório reúne estudos e anotações sobre os fundamentos do sistema
operacional Linux, servindo como base para administração de sistemas,
infraestrutura e ambientes de nuvem.

## Objetivo
Construir uma base sólida em Linux, compreendendo conceitos essenciais
que são pré-requisitos para trabalho com servidores, redes, segurança
e plataformas de cloud computing.

## Conteúdos abordados
- Estrutura de diretórios do Linux
- Usuários e grupos
- Permissões de arquivos e diretórios
- Comandos básicos de navegação e manipulação
- Processos e recursos do sistema

## Abordagem
- Foco em entendimento conceitual
- Execução prática em laboratório local
- Uso consciente de comandos
- Documentação clara e objetiva

## Importância para Infraestrutura
O domínio dos fundamentos do Linux é indispensável para qualquer profissional
que atue com infraestrutura, DevOps ou cloud, pois garante segurança,
estabilidade e previsibilidade nos ambientes.
